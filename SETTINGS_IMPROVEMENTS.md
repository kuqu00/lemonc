# lemonC 办公系统 - 设置功能完善总结

## 概述
本次更新完善了系统的设置功能,新增多项实用设置项,并优化了现有的设置结构。

## 新增设置功能

### 1. 客户跟进提醒
- **位置**: 功能设置 -> 提醒设置
- **功能**: 当客户超过设定天数未跟进时自动提醒
- **设置项**:
  - 启用/禁用客户跟进提醒
  - 跟进提醒天数 (默认7天,可设置1-90天)

### 2. 桌面通知
- **位置**: 功能设置 -> 桌面通知
- **功能**: 在桌面显示重要通知
- **通知类型**:
  - 待办到期和逾期通知
  - 合同即将到期通知
  - 系统重要事件通知
- **权限管理**: 自动请求浏览器通知权限

### 3. 自动锁定
- **位置**: 安全设置 -> 自动锁定
- **功能**: 无操作时自动锁定系统,保护数据安全
- **设置项**:
  - 启用/禁用自动锁定
  - 自动锁定时间 (默认30分钟,可设置5-120分钟)
- **实现**: 监听鼠标、键盘、滚动、触摸事件,自动更新活动时间

### 4. 默认标签设置
- **位置**: 数据管理 -> 默认标签设置
- **功能**: 自定义系统默认提供的标签选项
- **标签类型**:
  - 默认待办标签: 工作、客户、合同、其他
  - 默认客户标签: VIP客户、重点跟进、普通客户、潜在客户
- **操作**: 支持添加、删除标签

## 组件更新

### Settings.tsx
- 新增状态管理:
  - `desktopNotification`: 桌面通知开关
  - `followUpReminder`: 客户跟进提醒开关
  - `followUpDays`: 跟进提醒天数

  - `autoLockMinutes`: 自动锁定时间
  - `defaultTodoTags`: 默认待办标签数组
  - `defaultCustomerTags`: 默认客户标签数组

- 新增功能:
  - `handleRequestNotificationPermission()`: 请求桌面通知权限
  - 标签管理界面 (添加/删除标签)

- 移除内容:
  - 移除了 `isElectron()` 检查

### Layout.tsx
- 新增自动锁定功能:
  - 定时检查用户无操作时间
  - 超时自动登出
  - 监听用户活动事件 (鼠标、键盘、滚动、触摸)

- 更新默认设置初始化:

## 使用指南

### 启用桌面通知
1. 进入"系统设置" -> "功能设置"
2. 找到"桌面通知"卡片
3. 开启"启用桌面通知"开关
4. 系统会自动请求浏览器通知权限
5. 允许后即可在桌面接收通知

### 配置自动锁定
1. 进入"系统设置" -> "安全设置"
2. 找到"自动锁定"卡片
3. 开启"启用自动锁定"开关
4. 设置"自动锁定时间" (5-120分钟)
5. 无操作超过设定时间后自动登出

### 管理默认标签
1. 进入"系统设置" -> "数据管理"
2. 找到"默认标签设置"卡片
3. 点击"+ 添加"按钮添加新标签
4. 点击标签上的"×"按钮删除标签
5. 点击"保存设置"保存更改
